C Project Template
CProjectTemplate

Project template for C programming. Including debugging in VSCODE, makefile and Unit testing (+ Github CI)

Usage

  • make BUILD
  • make RUN
  • make TEST

Features

  • Clear structures
    • Source and test
    • Header file are organized under src/include/
    • Building with [makfile](makefile)
  • Unit testing
    • Organized by runtests.c
    • Use acutest.h debugging (Courtesy from mity/acutest)

Debugger with VScode

  • The configuration of debugger are supported under .vscode

Prerequist

  1. Vscode
  2. build-essentials
  3. Linux

How to use this C template

Build and Run with <tt>src/main.c</tt>

make BUILD_RUN

Unit testing

make TEST

Set workspace to this project

Use the debugger icon on VScode
